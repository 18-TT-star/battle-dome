# バトルドーム配布パッケージ

## 内容物
- バトルドーム（実行ファイル）
- README.txt（実行方法・注意事項）
- requirements.txt（依存パッケージ一覧・参考）

## 実行方法
1. Linux(x86_64)で `バトルドーム` に実行権限を付与してください。
   
   ```bash
   chmod +x バトルドーム
   ```
2. ターミナルで以下を実行
   
   ```bash
   ./バトルドーム
   ```

## 注意
- Pythonやcolorama等の依存はバイナリに同梱済みですが、
  システムに `libpython3.12` などが必要な場合があります。
- 動かない場合はエラーメッセージを確認し、必要なライブラリをインストールしてください。
- ソースコードはGitHubで公開中。
